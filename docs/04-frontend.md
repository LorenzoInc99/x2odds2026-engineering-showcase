# Frontend and Product Surface (Next.js)

## Role

Provide user-facing interaction patterns for:
- data exploration
- assistant interactions
- personalized views and saved context

## Implementation style

- Next.js App Router + API route handlers
- UI interacts with internal endpoints, not directly with private providers
- Server-side orchestration keeps sensitive operations out of client code

## Engineering value

- Full-stack integration between UI and AI/data backend
- Real request/response flow handling under evolving product requirements
- Feature delivery combined with operational maintenance
